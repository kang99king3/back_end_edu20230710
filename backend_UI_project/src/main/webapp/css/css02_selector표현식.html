文, 홍범도 등 흉상철거 논란에 "부디 숙고해주길"

文, 홍범도 등 흉상철거 논란에 "부디 숙고해주길"

문재인 전 대통령은 27일 국방부가 육군사관학교 내 홍범도 장군을 비롯한 독립군·광복군 영웅 5인의 흉상 이전을 추진하는 것과 관련 "깊은 우려를 표한다"고 밝혔습니다.

문 전 대통령은 이날 페이스북에 "대한민국의 뿌리가 임시정부에 있듯이 우리 국군의 뿌리도 대한독립군과 광복군에 있음을 부정하는 것인가"라며 "깊은 우려를 표한다"고 적었습니다.

그는 "국권을 잃고 만주로, 연해주로, 중앙아시아로 떠돌며 풍찬노숙했던 항일무장독립운동 영웅들의 흉상이 오늘 대한민국에서도 이리저리 떠돌아야겠는가"라며 지적했습니다.
